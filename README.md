# BookVault

A personal library manager where you can keep track of books you own, are reading, or want to read.

## Features

- Register and log in
- Add, edit, and delete books
- Each user only sees their own books

## Tech Stack

- Node.js / Express
- MongoDB / Mongoose
- Passport.js (local strategy)
- EJS templates

## Setup

1. Clone the repo
2. Run `npm install`
3. Create a `.env` file (see `.env.example`)
4. Run `npm run dev`
